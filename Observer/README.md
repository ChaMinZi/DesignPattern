# 옵저버(Observer) 패턴

소프트웨어 디자인 패턴 중 하나로 관찰자(Observer)라고 불리는 다른 객체 리스트와 이를 계속 추적하는 주체(Subject) 객체로 구성된 패턴입니다.
주체(Subject) 객체의 상태가 변하면, 모든 관찰자(Observer) 리스트들에게 이를 알립니다.
